# 搜索增强功能说明

## 概述

RGZW 管理后台系统在原有基础搜索功能之上，增加了高级搜索和过滤功能，以满足用户更复杂的数据检索需求。

## 功能特点

### 1. 高级搜索面板
- 可折叠的搜索面板，节省页面空间
- 多种搜索条件组合，提高检索精度
- 一键重置功能，方便快速清空搜索条件

### 2. 多维度搜索
- **关键词搜索**：支持在多个相关字段中进行全文检索
- **时间范围筛选**：可按时间段筛选数据
- **分类筛选**：提供多种下拉选项进行精确过滤
- **数值范围筛选**：支持年龄、金额等数值范围筛选

### 3. 智能匹配
- 支持不区分大小写的模糊匹配
- 多字段联合查询功能
- 搜索条件组合过滤

## 实现细节

### 学生管理页面高级搜索
**搜索字段包括：**
- 关键词：在姓名、学校、来源类型中搜索
- 时间范围：按创建时间筛选
- 性别：精确匹配男性或女性
- 年龄范围：通过滑块选择年龄区间（自动转换为出生日期范围）
- 状态：激活/未激活状态筛选
- 来源类型：精确匹配学生来源渠道

**技术实现：**
- 文件：`src/components/AdvancedSearch.vue`
- 集成：`src/views/students/Students.vue`
- 功能：条件构建 + API 查询

### 用户管理页面高级搜索
**搜索字段包括：**
- 关键词：在姓名、昵称、手机、邮箱中搜索
- 时间范围：按创建时间筛选
- 角色：精确匹配用户角色
- 状态：激活/未激活状态筛选
- 组织：精确匹配所属组织

**技术实现：**
- 文件：`src/components/AdvancedSearch.vue`
- 集成：`src/views/users/Users.vue`
- 功能：条件构建 + API 查询

## 使用指南

### 基础搜索
1. 在对应的管理页面使用基础搜索栏
2. 输入关键词或选择筛选条件
3. 点击"查询"按钮执行搜索

### 高级搜索
1. 点击"高级搜索"标题展开搜索面板
2. 根据需要填写高级搜索条件
3. 点击"搜索"按钮执行高级搜索
4. 如需清空所有搜索条件，点击"重置"按钮

## 技术架构

### 组件架构
```
AdvancedSearch.vue (高级搜索组件)
├── Collapse (折叠面板)
├── Form (搜索表单)
├── Input (关键词输入)
├── DatePicker (日期选择器)
├── Select (下拉选择器)
├── Slider (滑块选择器)
└── Button (操作按钮)
```

### 数据流程
```
用户输入 → 表单验证 → 条件构建 → API 请求 → 数据返回 → 表格渲染
```

### 查询条件构建
高级搜索将用户输入转换为后端可识别的查询对象：
- `$or` 操作符：用于多字段关键词搜索
- `$gte/$lte` 操作符：用于时间/数值范围筛选
- 直接匹配：用于精确值筛选

## 性能优化

- 搜索条件即时生效，无需手动刷新页面
- 合理的查询参数构建，避免过度查询
- 前端智能筛选，减少不必要请求

## 维护建议

1. **组件复用**：AdvancedSearch 组件设计为通用组件，可在其他页面复用
2. **扩展性**：组件结构支持便捷添加新的搜索条件
3. **兼容性**：保留基础搜索功能，保证向下兼容

## 注意事项

- 高级搜索面板默认收起，节省页面空间
- 搜索条件与基础搜索条件合并使用
- 重置功能会清空所有高级搜索条件，但保留基础搜索条件
- 年龄范围功能会自动转换为对应的出生日期范围进行查询
